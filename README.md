#How to install

Follow these steps:
1. Download jar file from https://github.com/ESOM-2017-1/stunning-fiesta/releases.
2. Copy the jar file to the Astah plugins folder(<user-home>/.astah/<Astah-edition>/plugins).
3. Start Astah.
